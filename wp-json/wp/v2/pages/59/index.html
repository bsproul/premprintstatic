Q:<\/strong> What type of file can I send to Premier Printing?
A: <\/strong>We recommend saving the file as a PDF. You may also send the file as AI, EPS or PSD. We prefer that you send the files with outlined fonts. These files are easy to handle and will likely speed up the turnaround time. Please DO NOT send Word document files.<\/p>\n\n\n\n

Q: <\/strong>How can I send my files to Premier Printing?
A: <\/strong>You can send us your files via email, or via courier. For PC users compress your file using ZIP, for Mac use Stuffit before sending your file to maintain integrity and reduce transfer time. Please email your file(s) to info@premierprintinglimited.com<\/a>.<\/p>\n\n\n\n
